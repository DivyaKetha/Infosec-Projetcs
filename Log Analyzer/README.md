# Log Analyzer (Cybersecurity Project)

## 📌 Description
This tool analyzes system logs to detect failed login attempts and identify suspicious IP addresses.

## 🚀 Features
- Detect failed login attempts
- Count attempts per IP
- Identify brute force behavior

## 🛠 Tech Used
- Python

## 📊 Example Output
⚠️ Suspicious IP: 192.168.1.30 → Attempts: 3

## 🎯 Use Case
Useful for SOC analysts to detect brute-force attacks.
